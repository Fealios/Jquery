Flashcards

Javascript

Javascript is a front end programming language, and the only programming language that Web Browsers automatically understand, allowing you to create and test in a basic environment with nothing more than a browser and a text editor.

Operators

Operators, as in math, are symbols that perform arithmetic functions on different data types of javascript.

Variables

A data item that may take on more than one value during the runtime of a program.

Variable Naming Conventions

Variable names should start with a letter and be in lower camel case.

Functions

A named section of a program that performs a specific task. In this sense, a function is a type of procedure or routine.

Methods

A method is a programmed procedure that is defined as part of a class and included in any object of that class.

Arguments

The actual input passed into a function.

Parameters

A variable as found in the function definition.

Return

A return statement is the information that part of a computer program gives to another part. In object-oriented programming, the term return is used by methods to perform an output.

Chaining methods

A common syntax for invoking multiple method calls in object-oriented programming languages. Each method returns an object, allowing the calls to be chained together in a single statement without requiring variables to store the intermediate results.

Strings

A string is any finite sequence of characters (i.e., letters, numerals, symbols and punctuation marks).

Booleans

The Boolean data type is a data type, having two values (usually denoted true and false), intended to represent the truth values of logic and Boolean algebra.

Undefinied

An undefined value is a condition where an expression does not have a correct value, although it is syntactically correct.

NaN

NaN, standing for not a number, is a numeric data type value representing an undefined or unrepresentable value, especially in floating-point calculations.

Escape

A key (frequently labeled Esc) found on most computer keyboards and used for any of various functions, as to interrupt or cancel the current process or running program, or to close a pop-up window. Expand. Also called Escape, Esc key.

Alert

Alerts display a dialog box with pretermined content.

Comments

A comment is a programmer-readable explanation or annotation in the source code of a computer program. They are added with the purpose of making the source code easier for humans to understand, and are generally ignored by compilers and interpreters.

jQuery

jQuery is a JavaScript library that allows web developers to add extra functionality to their websites. Once the jQuery library is loaded, a webpage can call any jQuery function supported by the library. Common examples include modifying text, processing form data, moving elements on a page, and performing animations.

Attributes

An attribute is a specification that defines a property of an object, element, or file. It may also refer to or set the specific value for a given instance of such.
